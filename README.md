# csp-lang-switch

Переключатель языка для **Clip Studio Paint 5.0.0** между английским и русским. Только для Windows.

## Что это

Clip Studio Paint не поддерживает русский язык. В программе нет русской локализации, и официально её добавить нельзя.

Эта программа переводит интерфейс Clip Studio Paint: меню, окна, диалоги и фильтры. Перевод сделан сообществом и встроен в один файл. Больше ничего скачивать или устанавливать не нужно.

Названия материалов в палитре не переводятся. Мы их специально не трогаем: правка каталога материалов может привести к тому, что материалы пропадут из программы.

## Как это работает

Программа подменяет английские файлы Clip Studio Paint на русские. Вы в любой момент можете переключиться обратно на английский. Оригинальные файлы сохраняются и восстанавливаются.

Важно: внутри самой Clip Studio Paint язык интерфейса должен быть **English**. Русский перевод устанавливается на место английского.

## Как скачать и запустить

1. Скачайте файл `csp-lang-switch.exe` со [страницы релизов](https://github.com/datacurse/csp-lang-switch/releases/latest).
2. **Закройте Clip Studio Paint**, если она открыта.
3. Запустите скачанный файл двойным щелчком.
4. В открывшемся окне выберите **русский** или **английский** и нажмите «Применить». Windows один раз спросит разрешение на изменения. Согласитесь.
5. Откройте Clip Studio Paint. Если переключали на русский, убедитесь, что в настройках самой программы выбран язык **English**.

Готово. Чтобы вернуть английский язык, снова запустите программу и выберите английский.
