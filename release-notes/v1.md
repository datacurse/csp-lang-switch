Multi-version release of the CSP language switcher.

**What's new:**
- Supports Clip Studio Paint **5.0.0** and **5.0.4** in one executable
- Version dropdown in the GUI; auto-detects your installed CSP version
- Warning when the installed version is unsupported or does not match the selected pack
- Refreshed 5.0.4 Russian pack captured from current 5.0.4 stock

**Download:** `csp-lang-switch.exe` — standalone Windows executable (no Python required).

**Requirements:**
- Windows
- Clip Studio Paint 5.0.0 or 5.0.4
- Close CSP before switching languages
- UAC prompt when writing to Program Files

**Usage:**
- Double-click to open the GUI picker; confirm the detected CSP version
- Or run from a terminal: `csp-lang-switch.exe --csp-version 5.0.4 russian`

**Требования:**
- Windows
- Clip Studio Paint 5.0.0 или 5.0.4
- Закройте CSP перед переключением языка
- Запрос UAC при записи в Program Files

**Использование:**
- Дважды щёлкните, откройте окно и проверьте определённую версию CSP
- Или из терминала: `csp-lang-switch.exe --csp-version 5.0.4 russian`
