Переключатель языка для **Clip Studio Paint** между английским и русским. Только для Windows.

**Поддерживаемые версии:** 4.0.0, 4.2.0, 5.0.0, 5.0.2, 5.0.4.

**Скачать:** `csp-lang-switch.exe` ниже. Один файл, Python не нужен.

## Как пользоваться

1. **Закройте Clip Studio Paint.**
2. Запустите программу, выберите версию CSP и язык, нажмите «Применить».
3. В настройках CSP язык интерфейса должен быть **English**. Перевод ставится на место английского.

Чтобы вернуть английский, запустите программу снова и выберите английский.
