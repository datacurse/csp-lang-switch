Переключатель языка для **Clip Studio Paint** между английским и русским. Только для Windows.

**Поддерживаемые версии:** 4.0.0, 4.2.0, 5.0.0, 5.0.2, 5.0.4.

**Скачать:** [csp-lang-switch.exe](https://github.com/datacurse/csp-lang-switch/releases/download/v0/csp-lang-switch.exe). Один файл, Python не нужен.

## Что переводится

- Основной интерфейс: меню, окна, палитры, диалоги (~97% главного UI)
- Категории палитры материалов (Color pattern, Download, 3D и т.д.)
- Меню «Фильтр»: категории, названия фильтров и параметры (37 plug-in DLL)
- Палитра инструментов: Pen, G-pen, Sketch и др.
- Дополнительные UI-модули: облако, каталог материалов, анимация, лицензия и др.

## Как пользоваться

1. **Закройте Clip Studio Paint.**
2. Запустите программу, выберите версию CSP и язык, нажмите «Применить».
3. В настройках CSP язык интерфейса должен быть **English**. Перевод ставится на место английского.

Чтобы вернуть английский, запустите программу снова и выберите английский.
