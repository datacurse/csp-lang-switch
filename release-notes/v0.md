Переключатель языка для **Clip Studio Paint 4.2.0, 5.0.0, 5.0.2 и 5.0.4** между английским и русским. Только для Windows.

**Скачать:** `csp-lang-switch.exe` — один файл, Python не нужен.

## Как скачать и запустить

1. Скачайте `csp-lang-switch.exe` ниже (или со [страницы релизов](https://github.com/datacurse/csp-lang-switch/releases/tag/v0)).
2. **Закройте Clip Studio Paint**, если она открыта.
3. Запустите скачанный файл двойным щелчком.
4. Выберите **версию CSP** (программа определит установленную автоматически), затем **русский** или **английский** и нажмите «Применить». Windows один раз спросит разрешение на изменения. Согласитесь.
5. Откройте Clip Studio Paint. Если переключали на русский, убедитесь, что в настройках самой программы выбран язык **English**.

Готово. Чтобы вернуть английский, снова запустите программу и выберите английский.

Важно: внутри Clip Studio Paint язык интерфейса должен быть **English**. Русский перевод устанавливается на место английского.
