Updated release of the CSP language switcher.

**What's new:**
- Simplified Russian-only picker: choose Russian or English in one list
- Switches main UI and plug-ins together (no subsystem checkboxes)
- Marks the currently active language; clearer status and restart message

**Download:** `csp-lang-switch.exe` — standalone Windows executable (no Python required).

**Requirements:**
- Windows
- Clip Studio Paint 5.0.0
- Close CSP before switching languages
- UAC prompt when writing to Program Files

**Usage:**
- Double-click to open the GUI picker
- Or run from a terminal: `csp-lang-switch.exe russian`, `csp-lang-switch.exe english`, `csp-lang-switch.exe status`

- Закройте CSP перед переключением языка
- Запрос UAC при записи в Program Files

**Использование:**
- Дважды щёлкните, чтобы открыть окно выбора
