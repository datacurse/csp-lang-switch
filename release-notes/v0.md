First public release of the CSP language switcher.

**Download:** `csp-lang-switch.exe` — standalone Windows executable (no Python required).

**Requirements:**
- Windows
- Clip Studio Paint 5.0.0
- Close CSP before switching languages
- UAC prompt when writing to Program Files

**Usage:**
- Double-click to open the GUI picker
- Or run from a terminal: `csp-lang-switch.exe russian`, `csp-lang-switch.exe english`, `csp-lang-switch.exe status`
